# 笔记管理器

一个用于创建、编辑和删除笔记的简洁 Web 应用。所有数据都会通过 `localStorage` 保存在浏览器中。

## 项目概览

笔记管理器可以快速记录带标题和可选内容的笔记。笔记完全存储在浏览器中，不需要账号或后端服务器。每条笔记都会记录创建时间和最后更新时间。

## 环境要求

- Node.js 18 或更高版本
- npm 9 或更高版本

## 安装

进入```anaconda prompt```，进入```cs146s```环境，并进入```bolt_react_notes```所在路径

```bash
npm install
```

## 本地运行

```bash
npm run dev
```

在浏览器中打开终端显示的地址，通常是 http://localhost:5173。

构建生产版本（基本不需要）：

```bash
npm run build
```

## 技术栈

- **React 18**：UI 库
- **TypeScript**：类型安全
- **Vite**：构建工具和开发服务器
- **Tailwind CSS**：工具类优先的样式框架
- **Lucide React**：图标
- **localStorage**：客户端持久化

## 功能

- 创建笔记，标题必填，内容可选
- 编辑已有笔记
- 删除笔记前显示确认提示
- 校验标题不能为空或仅包含空白字符
- 显示每条笔记的创建时间和更新时间
- 响应式布局，适配移动端和桌面端
- 通过 `localStorage` 在浏览器刷新后保留数据

## 已知限制

- 数据只保存在当前浏览器的 `localStorage` 中。清除浏览器数据或更换浏览器/设备会导致笔记丢失。
- 没有身份认证或多用户支持。
- 没有搜索、标签或分类功能。
- `localStorage` 通常有 5 MB 左右的容量限制，会限制可保存的笔记总量。
